## 2. 신규 기능: 해피엔딩 시나리오

기존 기획에 없던 **해피엔딩** 흐름이 추가되었습니다.

### 2.1 트리거 조건

**일정 점수 도달 시 해피엔딩 진입**. 해피엔딩 트리거 점수는 어드민 페이지에서 변경 가능.

### 2.2 해피엔딩 연출 흐름

```
게임 진행 중 (모든 장애물 클리어)
  │
  ▼
장애물이 더 이상 등장하지 않음
  │
  ▼
화면 오른쪽에 흰색 쥐(여친 쥐)가 가만히 서 있음
  │
  ├─ 기존: 주인공 제자리 달리기 + 장애물이 오른쪽→왼쪽 이동
  └─ 변경: 주인공이 오른쪽으로 달려감 + 흰색 쥐는 가만히 있음
  │
  ▼
주인공이 흰색 쥐에게 도달하여 부딪힘
  │
  ▼
두 쥐의 눈이 하트(♥)로 변함
  │
  ▼
엔딩 타이틀 "Love wins all" 표시 + 다시하기 아이콘(↻)
```

### 2.3 해피엔딩 후 다시하기

- 다시하기 아이콘 클릭 시:
  - "Love wins all" 텍스트 즉시 사라짐
  - 주인공이 왼쪽에서 새로 달려나옴
  - 오른쪽의 캐릭터들(여친 쥐 등)은 같은 속도로 오른쪽 화면 밖으로 이동하며 사라짐

## 3. 변경: 사망/게임오버 연출

### 3.1 사망 애니메이션 (신규)

기존: 충돌 시 즉시 정지 → "G A M E O V E R" 텍스트 표시

변경:
```
장애물 충돌
  │
  ▼
쥐 멈춤 + 표정/포즈 변화 (죽은 쥐 스프라이트)
  │
  ▼
쥐와 장애물이 함께 잠시 공중에 뜸
  │
  ▼
아래로 떨어지며 사라짐
  │
  ▼
"Game Over" 표시 + 다시하기 아이콘(↻)
```

### 3.2 게임오버 후 다시하기

- 다시하기 아이콘 클릭 시:
  - "Game Over" 텍스트 사라짐
  - 쥐가 왼쪽에서 다시 달려나옴

## 6. 변경: 게임 플레이 상세

### 6.1 점수 증가 방식

- 기존: 1점 단위 (크롬 다이노 기본)
- 변경: **50점 단위로 증가**

### 6.2 점프 입력 영역

- 기존: 게임 캔버스 영역 클릭/탭
- 변경: **페이지 전체 영역** 클릭/탭 시 점프 (게임 화면 + 푸터 영역까지)

> 클라이언트 원문: "게임 진행 중에는 페이지 모든 영역 누르면 쥐 점프 (게임 화면 + 푸터영역까지)"

### 6.3 속도 증가

- 클라이언트 원문: "속도, 빨라지는 정도는 레퍼런스 게임과 동일하게 가능할지?"
- → 크롬 다이노 게임의 속도 증가 패턴을 그대로 적용하되, 점수 단위만 50으로 변경

## 7. 삭제: 순위작성(점수 제출) 기능

순위작성 기능 전체를 삭제합니다.

**삭제 항목**:
- 게임오버 화면의 '순위작성' 버튼
- 연락처 입력 폼
- 개인정보 수집 동의 절차
- `POST /scores` 유저 점수 제출 API
- 유저 정보 sessionStorage 저장/복원
- 연락처 실시간 포맷팅/검증

**게임오버/해피엔딩 화면**에는 다시하기 아이콘(↻)만 표시.

## 9. 변경: 어드민 페이지

### 9.1 기존 기능 (유지)

- 비밀번호 기반 로그인
- 랭킹 조회/수정/삭제
- 로그인 시도 횟수 제한, 자동 로그아웃

### 9.2 신규 기능: 이미지 업로드 관리

어드민 페이지에서 각 캐릭터/장애물 이미지를 업로드하여 교체할 수 있습니다.

| 업로드 항목 | 파일 수 | 픽셀 제한 | 비고 |
| ---------- | ------- | --------- | ---- |
| 달리는 쥐 | 2장 | 176 x 192 (2배) | 2프레임 루프 |
| 점프 쥐 | 2장 | 176 x 192 (2배) | 2프레임 루프 |
| 죽은 쥐 | 2장 | 176 x 192 (2배) | 2프레임 루프 |
| 해피엔딩 쥐 | 2장 | 추후 확정 | 눈이 하트 |
| 여친 쥐 (대기) | 2장 | 추후 확정 | 해피엔딩 시 등장 |
| 해피엔딩 여친 쥐 | 2장 | 추후 확정 | 눈이 하트 |
| 장애물 1~5 | 각 2장 | 추후 확정 | 지면 장애물 |
| 나는 장애물 | 2장 | 168 x 120 (2배) | 공중 장애물 |
| 지면 타일 | 1장 | 4800 x 48 (2배) | 가로 반복 스크롤 |

- 업로드 시 픽셀 크기 검증 (지정 크기와 불일치 시 업로드 거부)
- WebP/PNG 포맷만 허용
- 업로드 즉시 반영 (프론트엔드 재배포 불필요)

> 기존 방식(파일 직접 교체 + 재배포)에서 어드민 UI 기반 교체로 변경됨.

### 9.3 신규 기능: 해피엔딩 점수 관리

- 해피엔딩 트리거 점수를 어드민 페이지에서 조회/변경 가능
- 예: 기본값 10,000점 → 어드민에서 5,000점으로 변경 시 즉시 반영
